python3 中字符串的方法总结

字符串方法总结

方法 具体用法
+ 字符串的连接
* 字符串相乘 => ‘fe’*2 = ‘fefe’
[] 通过索引获取 => ‘esf’[1] = ‘s’
[:] 通过索引截取 => ‘qwerty’[1:3] = ‘we’
in 判断存在返回布尔值
not in 判断不存在返回布尔值
r/R 原始字符串 - 原始字符串：所有的字符串都是直接按照字面的意思来使用，没有转义特殊或不能打印的字符。 原始字符串除在字符串的第一个引号前加上字母”r”（可以大小写）以外，与普通字符串有着几乎完全相同的语法
% 格式化字符串
