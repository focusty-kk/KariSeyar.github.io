javascrip中Function.prototype.bind()方法的理解

bind() 方法的主要作用就是将函数绑定至某个对象，bind() 方法会创建一个函数，函数体内this对象的值会被绑定到传入bind() 函数的值。

此时的this 指向的是window对象 

这个时候在o.f初始化的时候会把this(此时的作用域是o对象生成一个新的函数)

 更普遍的使用情形
function f(y,z){
return this.x+y+z;
}
var m=f.bind({x:1},2); 
console.log(m(3));  // 6

### bind背后的原理

Function.prototype.bindTest = function(scope){
    var fn = this;
    return function(){
        return fn.apply(scope,arguments);
    }
}
测试的结果：
var foo = {x: "Foo "};
var bar = function (str) {
    console.log(this.x+(arguments.length===0?'':str));
};
bar();                                   // undefined
var testBindBar = bar.testBind(foo);     // 绑定 foo
testBindBar("Bar!");                     // Foo Bar!
